CIBERSEGURIDAD

La ciberseguridad o seguridad informática es la suma de procesos y herramientas que se emplean para proteger anticipadamente o defender de ciberataques a dispositivos, sistemas electrónicos, servidores redes y programas, de posibles ataques digitales.

Tipos de ciberseguridad

Seguridad de la red Es la protección para los equipos y dispositivos conectados a una red. Los equipos de TI utilizan las tecnologías de seguridad de red, como los firewalls y el control de acceso a la red para regular el acceso de usuarios y administrar los permisos para recursos digitales específicos.
Seguridad de la nube Describe las medidas que toma una organización para proteger los datos y las aplicaciones que se ejecutan en la nube. Es importante para reforzar la confianza del cliente, proteger las operaciones tolerantes a errores y cumplir con la normativa de la privacidad de datos en un entorno escalable.
Seguridad de los datos Protege los datos en tránsito y en reposo con un sistema sólido de almacenamiento y una transferencia de datos segura. Los desarrolladores usan medidas protectoras, como el cifrado y las copias de seguridad aisladas, para la resistencia operativa frente a posibles brechas de datos.
Seguridad de las aplicaciones Es un trabajo coordinado para fortalecer la protección de una aplicación frente a la manipulación no autorizada durante las etapas de diseño, desarrollo y prueba. Los programadores de software escriben códigos seguros para evitar errores que puedan aumentar los riesgos de seguridad.

Tipos de ataques informáticos

Los ataques informáticos han ido en aumento en los últimos años, debido al mayor uso que se le da a internet. Los tipos más destacados son:

  1. Virus: se trata de un código maligno que infecta todos los archivos que posee un sistema. Para que se ejecute es necesario que un usuario lo ponga en funcionamiento. Al comenzar a funcionar, este código se esparce por todo el sistema y aquellos elementos a los que la cuenta tenga acceso.
  2. Malware: este término se utiliza para hacer referencia a cualquier software maligno que se crea con el objetivo de poder infiltrarse en el sistema y dañarlo.
  3. Gusanos: son programas que, después de infectar el equipo, realizan diversas copias de sí mismos y son difundidas por la red. Estos son difíciles detectar debido que su objetivo es el de difundirse e infectar numerosos equipos, por lo que no afectan el funcionamiento del sistema
  4. Adware: su principal función es la de mostrar anuncios de forma masiva. El objetivo es la de recaudar información y transmitirla para estudiar el comportamiento del usuario y dirigir mejor el tipo de publicidad.
  5. Spyware: es un programa espía, cuyo objetivo es el obtener información. Trabaja de forma silenciosa para poder recaudar toda la información posible sin levantar sospechas.

Mecanismos de protección y defensa

Ante los posibles ataques podemos protegernos mediante mecanismos de defensa, entre los más importante destacan:

  • Limitar el acceso a la información.
  • Crear contraseñas seguras y dimámicas.
  • Proteger el correo electŕonico.
  • Realizar copias de seguridad.
  • Cerrar las sesiones después de utilizar cualquier aplicación.
